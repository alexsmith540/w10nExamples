The data in this particular example (despite displaying "backward") was found using a prototype that I'll soon be releasing to the public. In the perfect world, all DAAC's wrap their datasets in a JPL developed, open-source specification, Webification (w10n), which is implemented by Taiga, an open-source software, to allow developers their own data exposure via a well formed URL (see an example at http://podaac-w10n.jpl.nasa.gov).

Since I had a w10n Amazon Cloud Machine (sweet I made an AMI: ami-a84dd698) running webification + NCAR Command Language binaries, I could not resist plugging in the raw data into my visualization-as-a-service widgets. If you look at the embed syntax below, the 3d widget is populated in the website straight from any data source's webified netcdf files, in this case subsetting inside of it and steppting through every 5th value.

All that and more, embedded in 2 lines of code in any website. In fact, the very same syntax that is driving this page is below.

How do we do this?

First, embed the libs in your site

<script data-main="//cds.jpl.nasa.gov/widgets/plot/1.0/js/main.js" src="//cds.jpl.nasa.gov/widgets/plot/1.0/components/requirejs/require.js"></script>

Now, business as usual:

/*Notice the colorbar= parameter. (http://scifari.org/colortable/matplotlib-1.2.1/rainbow/16[]?output=json). In this case, we are drawing in a matplotlib color schema from a 3rd party service. You can use the service too, please browse colors here: http://scifari.org/colortable/matplotlib-1.2.1/?output=json and some examples: http://scifari.org/colortable/
If you wish to, pass in a custom colorbar as a comma separated string of hex colors (i.e. "#ff0000,#00ff00,#00000ff")*/ 


<data-viz-type="plot.xyz" 
xdata="http://ec2-54-200-79-78.us-west-2.compute.amazonaws.com/data/MOD021KM.A2013311.1345.005.NRT.nc/Longitude[0:406,0:271]" 
xmask="[100000002004087730000]" 
xlabel="lon" 
ydata="http://ec2-54-200-79-78.us-west-2.compute.amazonaws.com/data/MOD021KM.A2013311.1345.005.NRT.nc/Latitude[0:406,0:271]" 
ylabel="lat" 
zdata="http://ec2-54-200-79-78.us-west-2.compute.amazonaws.com/data/MOD021KM.A2013311.1345.005.NRT.nc/EV_1KM_Emissive[15:16,0:2030:5,0:1354:5]" 
zmask="[-32767,-1]" 
zlabel="MOD021KM.A2013311.1345.005.NRT.nc/EV_1KM_Emissive, band 36" 
class="widgetxyz" 
format="flat"
width="500"
height="300"
colorbar="http://scifari.org/colortable/matplotlib-1.2.1/rainbow/64[]?output=json"></div>

**note: width and height can (and likely should) be set using native CSS. If you're not sure what the heck CSS is, then use width="" and height="" in your tag like above...
